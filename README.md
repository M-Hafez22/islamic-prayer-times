# Islamic Prayer Times

> A website provides You with **Prayer Times**  for one day depending on your time zone

---
 **Version 1.0.0**
---

## Features
	1. Current time ( in 12 formats)
	2. Hijri Date
	3. The prayers times for one day
	4. Remaining time for the next prayer
